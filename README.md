# Discography

## Installation

Run 

```bash
mix deps.get
```
# Initial setup

Create a .env file in the root folder

```dotenv
export SPOTIFY_CLIENT_ID="<<client_id>>"
export SPOTIFY_CLIENT_SECRET="<<client_secret>>"
```

Then run `source .env` every time this is modified to load them onto your OS.

# Start the app:

```bash
iex -S mix
```
